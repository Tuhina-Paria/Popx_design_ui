

This repository contains the implementation of a React JS front-end project based on a provided Adobe XD design. This project was developed as part of a qualifier task for an internship.

📌 Task Overview
Objective: Convert the given Adobe XD design into a fully functional React JS web application.
Requirements:
Use React JS for building the project.
Follow pixel-perfect design from the XD file.
Ensure the mobile app interface is centered on the webpage.
Implement smooth and functional navigation between pages.
Code should be clean, well-formatted, and maintainable.
Host the final project on Vercel or any other hosting service.

🚀 Project Structure
fronted/
├─ public/
├─ src/
│  ├─ components/        # Reusable React components
│  ├─ pages/             # Individual page components
│  ├─ App.jsx            # Main React component
│  ├─ main.jsx          # Entry point
│  └─ index.css          # Tailwind CSS and global styles
├─ package.json

⚡ Tech Stack
React JS – Frontend library
Tailwind CSS – Styling framework for rapid UI development
Vite – React build tool and development server
PostCSS & Autoprefixer – For processing Tailwind CSS
Vercel – Hosting platform

📂 Installation & Setup
Clone the repository
git clone https://github.com/Tuhina-Paria/Popx_design_ui
cd fronted
Install dependencies
npm install
Run the development server
npm run dev
Open http://localhost:5173
 to view the project in the browser.
Build for production
npm run build

The production-ready files will be generated in the dist/ folder.
